ngrx install
-------------------------------------------------------------------------------------------


install ngrx-schematics
-------------------------------------------------------------------------------------------------------
npm install @ngrx/schematics --save-dev

config ngrx-schematics as default schematics
-----------------------------------------------------------------------------------------------------
ng config cli.defaultCollection @ngrx/schematics


initial store and effects
-----------------------------------------------
ng generate store State --statePath=state --root -m=app.module.ts --collection @ngrx/schematics && ng generate effect state/App --root --module app.module.ts --collection @ngrx/schematics

generate users feature (folder:users, model:User)
-------------------------------------------------
ngm modulename in singular

ng g feature featurename/state/featurname -g=true -m=featurename
ng g feature courses/state/courses -g=true -m=courses

auth
-----------------
ngma
ng g feature auth/state/auth --flat=true -m=auth
